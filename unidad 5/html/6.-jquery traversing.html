siblings, next, nextAll, nextUntil, prev

siblings gets all the siblings of an element

next gets the immediately following sibling

nextAll gets all following siblings of the selected element

nextUntil gets all the following siblings up to but not including the element matched by the selector.

prev gets the immediately preceding sibling

prevAll gets all preceding siblings of the selected element

prevUntil gets all the preceding siblings up to but not including the element matched by the selector
